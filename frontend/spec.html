URAAAS

You are awesome as a Service

Please see README for use.

v0.0.1

Introduction

URAAAS (You Are Awesome As A Service) provides a modern, RESTful, scalable solution to the common problem of telling people that they are awesome.

Content Negotiation

URAAAS will respond to the following 'Accept:' values with appropriate content

  • text/plain - Content will be returned as a plain string.
  • application/json - Content will be returned as a JSON object { message: 'message', subtitle: 'subtitle' }
  • text/html - Content will be returned as an HTML page with a twitter bootstrap jumbotron , containing the message and the subtitle.

API

/awesome/:from

Will return content of the form "You're awesome. - :from" e.g. /you/Tom will return "You're awesome. - Tom"

/awesome/:name/:from

Will return content of the form "Hey :name, you're awesome. - :from", e.g. /awesome/Tom/Chris will return "Hey Tom, you're awesome. - Chris"

/humanity/:from

Will return content of the form "You just revived my faith in humanity. - :name", e.g. /humanity/Tom will return "You just revived my faith in humanity. - Tom"

/yoda/:from

Will return content of the form "You're my personal yoda. - :name", e.g. /yoda/Chris will return "You're my personal yoda. - Chris"

/amiawesome

Will return content of the form "Hell yeah!"
